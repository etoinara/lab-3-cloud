Лабораторная работа: Создание и деплой веб-сайта
Название проекта
Мой первый статический веб-сайт

Описание
Данный проект представляет собой простую веб-страницу, созданную для демонстрации навыков верстки и деплоя статического контента в облачной среде. Приложение выполняет роль демонстрационной визитки.

Стек технологий
HTML5

CSS3

Git (система контроля версий)

Netlify (платформа для деплоя)

Инструкция по локальному запуску
Склонируйте репозиторий: git clone <ссылка-на-ваш-репозиторий>

Перейдите в папку проекта.

Откройте файл index.html в любом современном браузере.

Инструкция по деплою
Платформа: Netlify.

Настройки: Использовался метод прямого импорта из GitHub. Команда сборки (Build command) оставлена пустой, так как проект состоит из статических файлов. В качестве директории публикации (Publish directory) указан корень репозитория (.).
